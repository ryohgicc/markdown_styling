# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-24

### Added

- Initial single-page Markdown poster editor
- Live preview for Markdown content
- Three built-in poster themes
- Multiple canvas presets for poster rendering
- PNG export using browser-side DOM capture

### Improved

- Full-length poster export flow for long content
- Export width handling for code blocks
- Export table rendering to avoid preview-style overflow behavior
- Export height measurement for long tables and complex content blocks

### Notes

- Current version is focused on static frontend usage
- External libraries are loaded through CDN
